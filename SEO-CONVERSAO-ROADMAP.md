# Mapa de SEO e Conversao - New Age Embalagens

Objetivo: transformar o site em referencia para empresas que precisam escolher, especificar e cotar caixas de papelao ondulado.

Principio central: cada pagina deve responder uma duvida real de compra e levar o usuario para uma rota clara:

- calculadora tecnica
- catalogo FEFCO
- pedido tecnico direto
- WhatsApp comercial

## Estrutura Padrao de Cada Pagina

Cada pagina nova deve seguir esta estrutura:

1. Hero com H1 direto, promessa clara e 2 CTAs.
2. Bloco "quando usar" com linguagem comercial.
3. Bloco tecnico curto com onda, ECT, BCT, papel e gramatura quando aplicavel.
4. Bloco de modelos FEFCO relacionados.
5. Bloco de perguntas frequentes.
6. CTA final para calculadora, WhatsApp ou pedido tecnico.
7. Schema adequado.

## Regras de Linkagem Interna

- Toda pagina tecnica deve linkar para `/orcamento-rapido/#configurador`.
- Toda pagina FEFCO deve linkar para o catalogo e para pedido tecnico.
- Toda pagina por segmento deve linkar para guia de onda/ECT e para WhatsApp.
- A home deve linkar para as 3 principais rotas: calculadora, catalogo FEFCO e pedido tecnico.
- O bot de WhatsApp deve ser tratado como CTA comercial, nao como suporte generico.

## CTAs Padrao

- `Usar calculadora tecnica`
- `Solicitar cotacao tecnica`
- `Falar com comercial no WhatsApp`
- `Comparar modelos FEFCO`
- `Enviar pedido tecnico`
- `Desenvolver caixa sob medida`
